Cuny Replica

This app would be a remake of the current cuny system as it is not synced well with other Servers. It utilizes Ruby on Rails for its backend API. The web app allows external financial servers to communicate with it.

DEK

This app allows users to create their own unique stack of flash cards to aid in the learning process. The web app uses a spaced-repetition style of learning to determine an optimized quiz for Education for a particular user.

My Assistant

This app uses machine learning with weather as inputs and clothing categories as outputs. The app stores the inputs and outputs in a postgres database.

Bomb-wars

This game allows two players to play on a single keyboard to try and destroy all enemies including the other player with bombs. All players are from the same class type using an object oriented approach to the game.

Gears

This web app allows the user to enter peg distances and a start to end ratio to allow for automatically generating appropriate gear sizes. The app uses mathematic matrices to achieve the computations for the gear ratios.
