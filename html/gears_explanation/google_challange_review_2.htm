1. I was given an example case of having pegs with distance [4, 30, 50]
    -4------30---------------50---------
  2. I was given the solution as the first gear being radius 12, and the last gear being radius 6 ie: 1/2 of first.
  3. I used 30-4=26 and 50-30=20 to try and find a pattern and inserted the given answer into the equation to find: 26=12+14 and 20=14+6.
  4. noticing the number 14 occured two times, I made that into a constant value coming up with 26=12+d and 20=d+6 or even more general: a = c+d and b = d+e

  5. Next I set the first and last radii to half of eachother resulting in: a = c+d and b = d+c/2
  6. This was a system of equations, and we solve those via matrix rref.
  7. The matrix became:
    c d a
    d c/2 b
    0 0 0
  8. Substituting becomes:
    1 1 26
    1 1/2 20
    0 0 0
  9. Solving the matrix yeilds:
    1 0 12
    0 1 14
    0 0 0
    And as we know the last radius is simply 1/2 of the first yeilding 12 and 6
  10. Keep in mind it is not always possible to generate the correct result.
  11. Simplified Matrix:
    1 1 a
    1 1/2 b
  12. I proceeded to extrapolate the data with result:
    1 1 0 0 0 a
    0 1 1 0 0 b
    0 0 1 1 0 c
    0 0 0 1 1/2 d
    0 0 0 0 0 0
    for 5 pegs
  13. Then I programed the solution in python and finished the problem.
